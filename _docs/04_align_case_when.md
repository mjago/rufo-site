### unnamed 13
```ruby
# GIVEN
case
 when 1 then 2
 when 234 then 5 
 else 6
 end
```
```ruby
# BECOMES
case
when 1 then 2
when 234 then 5
else 6
end
```
```ruby
# with setting `align_case_when true`
case
when 1   then 2
when 234 then 5
else          6
end
```
